<br/>

### Development
```bash
git clone git@github.com:Snapaper/snapaper-react.git
yarn install
```

```bash
// Development Server
yarn run dev
```

```bash
// Production Build
yarn run build
```

```bash
// Static Website
yarn run export
```

```bash
// Server-Side Rendering
yarn run start
```
